## Design Notes

### Wave Progression

Stressors unlock progressively to teach increasingly complex stress management:

* **Waves 1-3:** Intrusive Thought only â€” Learn basics with foundational stressor
* **Waves 4-5:** + Time Pressure â€” Introduce urgency mechanic
* **Waves 6-8:** + Environmental Noise â€” Add distraction/wobble pattern
* **Waves 9-10:** + Expectation â€” Introduce orbit/dash pattern

* **Waves 12:** + Self-Doubt â€” Introduce ability debuff mechanic
* **Waves 13-14:** + Overwhelm â€” Add acceleration/cascade mechanic
* **Wave 15+:** All 7 types â€” Full complexity, all mechanics active

### Movement Patterns

* **Wobble:** Environmental Noise (1 stressor)
